# Ardino-Xamarine-UdpListener
スマホでUDPパケットを受信。  
Arudino等で作れる簡易な電文の受信向け。


### 機能

![ClipToClip](http://saneyukim.github.io/images/CIMG3202.png)

青いクリップと黒いクリップの間の電圧がスマホで見れます！

### コンパイル

　ネットワークに極端な負荷をかけるソフトの為、インストール方法
は用意されていません。
