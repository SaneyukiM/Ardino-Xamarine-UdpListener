# Ardino-Xamarine-UdpListener
スマホでUDPパケット受信。 (Arudino等で作れる簡易な電文向け。)


### 機能

![ClipToClip](http://saneyukim.github.io/images/CIMG3202.png)

青いクリップと
	黒いクリップの間の電圧が！

	スマホで見れます！


### コンパイル

　ネットワークに極端な負荷をかける為のソフトなので、インストール方法
は用意されていません。
　ソースの内容を理解の上、自己責任で使用してください。
